By Haider Shawl

Learn how to make gorgeous websites without writing any CSS code.

Get Started

What you will do:
  • Learn how to use custom CSS styles in your website
Prerequisites:
  • HTML and CSS

Get Started

Create a new file, type the following code in it, save it, and then open the file in your browser.

<!DOCTYPE html>
<html>
<head>
  <link href="https://cdn.jsdelivr.net/npm/bootstrap@5.0.2/dist/css/bootstrap.min.css" rel="stylesheet" integrity="sha384-EVSTQN3/azprG1Anm3QDgpJLIm9Nao0Yz1ztcQTwFspd3yD65VohhpuuCOmLASjC" crossorigin="anonymous">
  <title> My First Website </title>
</head>
<body class="bg-light">
  <div class="container">
    <h1 class="text-primary text-center"> Haider </h1>
    <p>Hello World!</p>
  </div>
</body>
</html>

You will see that our website has CSS styling even though we didn't write a single line of CSS code! How did this happen?

Web developers don't write the entire CSS code themselves. Instead we use tools like Bootstrap, Materialize CSS etc. These tools allow us to use prewritten CSS styles in our HTML document very easily.

Let's see how we use Bootstrap:

The first step is to make Bootstrap's CSS code available to us. We can do this by simply adding the following lines inside the head tag of our document:

<link href="https://cdn.jsdelivr.net/npm/bootstrap@5.0.2/dist/css/bootstrap.min.css" rel="stylesheet" integrity="sha384-EVSTQN3/azprG1Anm3QDgpJLIm9Nao0Yz1ztcQTwFspd3yD65VohhpuuCOmLASjC" crossorigin="anonymous">
<script src="https://cdn.jsdelivr.net/npm/bootstrap@5.1.0/dist/js/bootstrap.bundle.min.js" integrity="sha384-U1DAWAznBHeqEIlVSCgzq+c9gqGAJn5c/t99JyeKa9xxaYpSvHU5awsuZVVFIhvj" crossorigin="anonymous"></script>

Now, we can start using Bootstrap's functionalities.

To apply bootstrap's styling to any element, we give the "class" attribute a specific value.

<p class="text-primary">Hello World!</p>

We will see that the color of the text changes to blue! It is that simple.

Some more bootstrap usage examples:

  1. Font size
<p class="fs-4">Hello World!</p>

Classes fs-1 to fs-6 set different font sizes to the element's text

  1. Font Alignment
<p class="text-center">Hello World!</p>

The text-center class allows you to set the alignment of the element's text.

  1. Button 
<button type="button" class="btn btn-primary">Click Me</button>

The btn and btn-primary classes give custom style (color and size) to any button.

You may go to the official website of Bootstrap to learn about all of their available styling options.
