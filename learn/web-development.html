  • Week 1 - Introduction

    Introduction

    The first terms you must know are server and client. Both are basically computers. The client is the device you are viewing the website on, and the server is the computer which is sending the website to your device through the internet.

    To access a website, you need a web browser like Google Chrome, Safari etc. When you enter a URL (like www.codekashmir.ml) in your web browser, your browser makes a request to the server which hosts the website. Then the server returns code which is interpreted by your browser. And thus, your browser shows the website to you.

    New Terms

    • Server - A server is a computer or system that provides resources, data, services, or programs to other computers(clients).
    • Client - A client is a personal computer or device that is capable of obtaining information and applications from a server.
    • Web Browser - A web browser is an application that retrieves the necessary content from a web server and then displays the page on the client's device.
    • Hypertext - Hypertext is text displayed on a computer display or other electronic devices with references to other text that the reader can immediately access.
    • Web Page - A web page is a hypertext document provided by a website and displayed to a user in a web browser.
    • Web Site - A website is a collection of web pages and related content.
    • World Wide Web - The World Wide Web—commonly referred to as WWW, W3, or the Web—is an interconnected system of public webpages accessible through the Internet.
    • HTML - HTML (HyperText Markup Language) is the most basic building block of the Web. It defines the meaning and structure of web content.
    • CSS - CSS (Cascading Style Sheets) is a language used to describe the presentation of a document written in HTML.
    • Syntax - Syntax is the set of rules that define what the various combinations of symbols mean. This tells the computer how to read the code.

    It is not necessary to memorize the definition of these terms right now. One amazing thing about the internet is that you can look up anything at anytime!

    Set Up

    To write your first program, you will only need a source code editor and a web browser on your computer. A source code editor is an application that is specialized for writing software. They make writing and reading the source code easier by differentiating the elements so programmers can more easily look at their code. My favorite source code editors are Microsoft Visual Studio Code and Sublime Text. You may also use Notepad on your computer (yes! you can write programs from notepad). You may use any browser except the Internet Explorer because it does not work with some modern web features.

    Now that we have our source code editor and web browser open, let's get started!

  • Week 2 - HTML

    Creating your first website!

    Create a new file in your text editor and save it by the name "website.html". Then type in the following code into the file and save it. Now go to the location where you saved the file and double click on it. Voila! You should see your first website, up and running!

    <!DOCTYPE html>
        <html>
          <head>
            <title> My First Website </title>
          </head>
          <body>
                <h1> Haider </h1>
                <p>Hello World!</p>
                <a href="https://www.codekashmir.ml">Click me!</a>
          </body>
        </html>

    Now let's see what we did here.

    HTML Syntax

    We used HTML to create a basic website.

    HTML code is written in the form of tags. Tags usually look something like this: <code>Some Text Here</code> . <code> is called the opening tag and </code> is called the closing tag. Tags are usually created in pairs. The text and other tags we want to display go in between the opening tag and the closing tag.

    Now let us discuss the some important HTML tags:

    • <!DOCTYPE html><html> ... </html> The "html" tag is used to tell the web browser that our file is a HTML file. All other tags go inside the "html" tag. Read more
    • <head></head> The "head" tag contains information about our HTML document. Read more
    • <title></title> The "title" tag lets you set the browser's title bar. Read more
    • <body></body> The "body" tag contains all the tags which will be displayed the HTML document in the web browser. Read more
    • <h1> Text </h1> The "h1" tag lets you write a heading . The "h" stands for heading and the "1" indicates the size of the text. Read more
    • <p> Text </p> The "p" tag lets you write a paragraph. Read more
    • <a href="https://www.codekashmir.ml"> Click me! </a> The "a" tag lets you to create a link in your website. The destination link is specified in the "href" attribute (more about attributes ahead). Read more
    • <img src="https://www.w3schools.com/images/lamp.jpg"> The "img" tag lets you to display images in your website. The URL of the image is specified in the "src" attribute. Read more
    • <div></div> The "div" tag lets you group many elements inside one element and work with them at the same time (eg. styling). This will make more sense in the CSS session. Read more

  • Week 3 - CSS

    Adding some style to our website!

    Open the file used in the previous week. Add the following code above the "title" tag and then refresh the page in the browser to see the changes.

    <style>
              h1 {
                color: red;
                font-size: 50px;
              }
        
              p {
                color: white;
              }
        
              body {
                background-color: blue;
              }
        </style>

    You must see a colorful website this time! How did this happen?

    This happened with the help of CSS (Cascading Style Sheets) language. It is a styling language which allows us to add designs and animations to our website.

    You many write your CSS code between "style" tags or as an HTML attribute or create another file for it. Learn more

    CSS syntax

    selector {
          property: value;
        }

    First we specify the element of our HTML document which we want to stylize. Then we write the property and value pairs inside the curly braces ({}). The property and value must be separated by a colon (:). A semicolon (;) must used at the end of each property-value pair.

  • Week 4 - Bootstrap

    Create a new file, type the following code in it, save it, and then open the file in your browser.

    <!DOCTYPE html>
        <html>
          <head>
            <link href="https://cdn.jsdelivr.net/npm/bootstrap@5.0.2/dist/css/bootstrap.min.css" rel="stylesheet" integrity="sha384-EVSTQN3/azprG1Anm3QDgpJLIm9Nao0Yz1ztcQTwFspd3yD65VohhpuuCOmLASjC" crossorigin="anonymous">
            <title> My First Website </title>
          </head>
          <body class="bg-light">
            <div class="container">
              <h1 class="text-primary text-center"> Haider </h1>
              <p>Hello World!</p>
            </div>
          </body>
        </html>

    You will see that our website has CSS styling even though we didn't write a single line of CSS code! How did this happen?

    Web developers don't write the entire CSS code themselves. Instead we use tools like Bootstrap, Materialize etc. These tools allow us to use prewritten CSS styles in our HTML document very easily.

    Let's see how we use Bootstrap:

    The first step is to make Bootstrap's CSS code available to us. We can do this by simply adding the following line in our document:

    <link href="https://cdn.jsdelivr.net/npm/bootstrap@5.0.2/dist/css/bootstrap.min.css" rel="stylesheet" integrity="sha384-EVSTQN3/azprG1Anm3QDgpJLIm9Nao0Yz1ztcQTwFspd3yD65VohhpuuCOmLASjC" crossorigin="anonymous">

    Now, we can start using Bootstrap's functionalities.

    To apply bootstrap's styling to any element, we give the "class" attribute a specific value.

    <p class="text-primary">Hello World!</p>

    We will see that the color of the text changes to blue! It is that simple.

    You may go to the official website of Bootstrap to learn about all of their available styling options.

    Its finally time that we deploy our website to the internet and show it to the world!

    To make our website available to the world, we will need a web hosting provider. I will be using the service of 000webhost for our purpose. It is easy to setup and completely free.

    First, you will need to create an account on 000webhost's website. Then you will be asked to provide details for a website name and a password. Provide your name as the website name and set a password. Then go to the manage website page and then click on file manager. It will take you to a new page where you can add and delete files on your server. Name your HTML file "index.html" and upload it inside the "public_html" folder. And that's it!
