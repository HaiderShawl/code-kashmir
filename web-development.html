Week 1

Introduction

The first terms you must know are server and client. Both are basically computers. The client is the device you are viewing the website on, and the server is the computer which is sending the website to your device through the internet.

To access a website, you need a web browser like Google Chrome, Safari etc. When you enter a URL (like www.codekashmir.ml) in your web browser, your browser makes a request to the server which hosts the website. Then the server returns code which is interpreted by your browser. And thus, your browser shows the website to you.

New Terms

  • Server - A server is a computer or system that provides resources, data, services, or programs to other computers(clients).
  • Client - A client is a personal computer or device that is capable of obtaining information and applications from a server.
  • Web Browser - A web browser is an application that retrieves the necessary content from a web server and then displays the page on the client's device.
  • Hypertext - Hypertext is text displayed on a computer display or other electronic devices with references to other text that the reader can immediately access.
  • Web Page - A web page is a hypertext document provided by a website and displayed to a user in a web browser.
  • Web Site - A website is a collection of web pages and related content.
  • World Wide Web - The World Wide Web—commonly referred to as WWW, W3, or the Web—is an interconnected system of public webpages accessible through the Internet.
  • HTML - HTML (HyperText Markup Language) is the most basic building block of the Web. It defines the meaning and structure of web content.
  • CSS - CSS (Cascading Style Sheets) is a language used to describe the presentation of a document written in HTML.

It is not necessary to memorize the definition of these terms right now. One amazing thing about the internet is that you can look up anything at anytime!

Set Up

To write your first program, you will only need a source code editor and a web browser on your computer. A source code editor is an application that is specialized for writing software. They make writing and reading the source code easier by differentiating the elements so programmers can more easily look at their code. My favorite source code editors are Microsoft Visual Studio Code and Sublime Text. You may also use Notepad on your computer (yes! you can write programs from notepad). You may use any browser except the Internet Explorer because it does not work with some modern web features.

Now that we have our source code editor and web browser open, let's get started!
